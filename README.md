
# Link Page

Página personal de links creada con HTML, CSS y JavaScript vanilla.

El contenido editable vive en `content.json`, así que puedes cambiar nombre,
foto, footer y enlaces sin tocar el HTML.

## Screenshots

![App Screenshot](https://github.com/Ericriera/MyPortfolio/blob/main/img/linkpage.png)

## Features

- Responsive design
- Links
- Contenido editable desde `content.json`

## Uso local

Sirve la carpeta con cualquier servidor estático:

```bash
python3 -m http.server 8080
```

Después abre:

- `http://localhost:8080/` para ver la página

Para editar la página, modifica `content.json` directamente.
